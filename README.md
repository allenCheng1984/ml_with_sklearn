## 環境與套件安裝

請先安裝好 Anaconda
ref: [Mac 安裝 Anaconda](https://medium.com/ayuth/install-anaconda-on-macos-with-homebrew-c94437d63a37)


安裝 Scikit-Learn
`conda install -c intel scikit-learn`

安裝相關套件
`conda install -c intel pandas pillow nltk`
